Predicted Useful Tags (Word Cloud)

    The word cloud represents topics generated using Latent Dirichlet Allocation (LDA) over top (votes) 1000 questions on each topic. The LDA model infers topic distribution on new, unseen documents [questions].
    The top 20 topics so generated are aggregated and assigned weights based on the probability distribution to form the word cloud.
    This step helps to identify topics that are closely related to a given tag but not being mentioned explicitly along with questions.

    Tag Relations

    The chord diagram represents a detailed correlation of a given tag with other tags in the network. The thickness shows the strength of the connection between tags. This diagram also visualizes how tags from different categories are interrelated.

    Compared to other related tags

    The column chart allows users to compare between different tags that belong to the same category. Tags can be compared based on the number of posts and accepted answers.
